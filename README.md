## Kotlin vs Java

| Kotlin                  | Java            |
| :----------------------:| :-------------: |
| ```object```            | ```static```  |
| ```var p = Person()```  | ```Person p = new Person()```  |
| ```for (i in array){print(i)} ``` | ```java ```  | 
|                                   |        `for (int i=0; i<array.length; i++){ `         |
| |                                    `print(i);` } |
